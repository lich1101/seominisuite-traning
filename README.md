# SEO Mini Suite - Drupal Project

## Mô tả
SEO Mini Suite là một dự án Drupal cung cấp các công cụ SEO toàn diện cho website. Dự án bao gồm theme tùy chỉnh và các module SEO chuyên dụng.

## Tính năng chính
- Theme tùy chỉnh Cassiopeia với giao diện hiện đại
- Công cụ phân tích SEO
- Tối ưu hóa từ khóa
- Báo cáo SEO chi tiết
- Giao diện responsive

## Cài đặt
1. Clone repository này
2. Cài đặt Drupal theo hướng dẫn chuẩn
3. Kích hoạt theme Cassiopeia
4. Cấu hình các module SEO

## Yêu cầu hệ thống
- PHP 7.4+
- MySQL 5.7+ hoặc MariaDB 10.3+
- Apache/Nginx
- Drupal 9+

## Cấu trúc thư mục
```
├── sites/
│   ├── all/
│   │   └── themes/
│   │       └── cassiopeia_theme/    # Theme tùy chỉnh
│   └── default/
├── modules/                         # Các module Drupal
├── themes/                          # Themes mặc định
└── includes/                        # File include của Drupal
```
